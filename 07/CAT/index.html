蔣勳《貓》

DSE 指定篇章重點筆記

學習目標：掌握「欲揚先抑」手法、理解對生命的尊重、分析作者情感變化。

1. 預習導入：情感脈絡

閱讀本文時，我們需要緊隨作者的「眼睛」，看看他眼中的貓發生了甚麼變化。請試著理解以下的情感流動：

既定印象陰沉、冷漠、像刺客
童年經歷被貓凝視、跺腳嚇貓
成年觀察家貓狡黠、野貓如鬼魅
關鍵轉折廢車房見證小貓掙扎求存
最終感悟體悟生命的艱難與尊嚴

2. 為什麼討厭貓？

作者在文章前半部（第1-8段）極力描寫對貓的反感。這是一種「欲揚先抑」的手法。


第一階段：直觀感受

  • 性格：陰沈、冷漠
  • 行為：像潛伏的刺客，無聲無息
  • 眼神：要攝去人的魂魄，令人不寒而慄
  • 對比：不像狗那樣忠誠、熱情

第二階段：生活體驗

  • 童年：故意跺腳嚇貓，貓卻反應冷淡
  • 成年：家貓雖然撒嬌，但眼神依舊狡黠
  • 野貓：在巷弄中如鬼魅般穿梭，令人恐懼

思考問題：作者為何要詳細描寫「跺腳嚇貓」這個片段？

3. 轉折點：廢棄車房的震撼

全篇高潮！

第13-24段是文章的轉折。作者偶然發現了廢棄車房裡的貓家族。

4. 感悟：生命的尊嚴

作者明白了甚麼？

經歷了車房一事，作者對貓的觀感發生了180度大轉變。

「我看到了一種不可侮辱的生命尊嚴。」

放下成見

不應以人類高高在上的視角審視動物

生存不易

即使在卑微的角落，生命也在努力活著

骨氣

不為食物而折腰，不願受人圈養

延伸思考：螞蟻與貓 (比較閱讀)

筆記中提到了「螞蟻」的故事。工蟻終日勞動，蟻王養尊處優。

共通點：

  • 對象都是微小的生物 (小生靈)
  • 都展現了作者對「弱勢/卑微」生命的關注
  • 都希望能有「和平的陽光」照進黑暗的角落

  • 2.

  • 4.
    2. 作者為何要選跺腳嚇貓的片段來寫？答：因為這片段反映了作者遇貓時的 反應(或恐懼)，為下文 ( 第 11 段 ) 作 伏筆 的作用。
  • 5.
    第6-8段：轉而寫 在家養 的貓，雖然家貓會撒嬌，但作者仍不愛貓，原因是貓兒太 狡黠，充滿機心。

    3. 根據第6-8段，作者不喜歡貓的原因：(1) 貓的眼神 狡黠、陰森。 (2) 貓的 叫聲淒厲。 (3) 貓 不像狗一樣忠誠。
